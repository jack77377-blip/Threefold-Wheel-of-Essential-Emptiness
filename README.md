# Gemini 教學網站 GitHub Pages 版

這是一個可直接部署到 GitHub Pages 的靜態網站。

## 使用方式

1. 打開 `index.html`
2. 找到「主要內容」區塊
3. 將 Gemini 分享頁面的內容貼入該區塊
4. 上傳到 GitHub Repository
5. 到 Settings → Pages 開啟 GitHub Pages

## 檔案說明

- `index.html`：網站主頁
- `style.css`：網站樣式
- `README.md`：部署說明

## GitHub Pages 設定

Source：Deploy from a branch  
Branch：main  
Folder：/root
