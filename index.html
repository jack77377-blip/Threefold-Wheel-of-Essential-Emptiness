Deploy

GitHub Pages 上架方式

01

建立 Repository

到 GitHub 建立新專案，例如 gemini-teaching-site。

02

上傳檔案

將 index.html、style.css、README.md 上傳到專案根目錄。

03

開啟 Pages

進入 Settings → Pages，Source 選擇 Deploy from a branch，Branch 選 main / root。
